# Spring + Hibernate: One-to-One Relationship

## Описание
Проект по интеграции Spring Framework и Hibernate ORM. 
Реализована связь One-to-One между сущностями User и Car с использованием HQL-запросов.

## Технологии
- Java 8
- Spring Framework 5.0.1
- Hibernate ORM 5.2.9
- MySQL 8+
- Maven
- JDBC

## Функциональность
1. Настройка Spring контекста с Java-based конфигурацией
2. Интеграция Hibernate через Spring ORM
3. Создание сущностей с аннотациями JPA
4. Реализация связи One-to-One между User и Car
5. CRUD операции через Hibernate Session
6. HQL-запросы для поиска по связанным сущностям
7. Управление транзакциями через @Transactional
